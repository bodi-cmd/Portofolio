ProBPM

- take care of your heart

ProBPM it's the bracelet that everyone needs.

It basically is a device, that can save your life, when you do a heart attack. This bracelet, is connected permanently to your smartphone via bluetooth, and it monitors your heart rate. When it detects a dramatic change, it will warn you, and if it detects that your heart is not beating anymore, it will call an ambulance with shared location.

Project Started On: 2 May 2018
Project Ended On: 20 May 2018
Project Production Costs: ~$20
Difficulty level: Medium

Let's start with the main sensor. It is a cheap chinese pulse sensor. I will use it in combination with an Attiny85 microcontroller, because it's cheap and powerfull enough. The conectivity part is ensured by a HC-05 Bluetooth Module. I'll leave you some pictures with these components below.

For powering the project, I chose one Li-Ion 18650 cell. It has a lot of power and capacity, and it's also very compact. The battery voltage it's between 3.7V and 4.2V, so I boosted this voltage to 5V using a USB Charging Module (it is usually used in mobile power banks). Now the electronic part is finished. You can see below the chematic.

For the enclosure, I designed and 3D printed a case, using AutoCAD

Now, the physical part is done. We need to program the Android App and the Attiny. For the Android App, I used java in Android Studio and for the microcontroller i used C++ in the Arduino IDE.
